Opis

Sadržaj drugog kolokvija

Strukture podataka

Drugi kolokvij sastoji se od razgranatih struktura koje smo radili u drugom dijelu semestra:

  • Stabla (BST)
  • Grafovi

Za rad sa stablima koristimo rekurzije u situacijama kad moramo obići cijelo stablo (npr. ispis). Međutim, za neke operacije (npr. dodavanje novog čvora) ili traženje po broju na kojem se temelji uređaj u stablu, rekurzije nisu učinkovite jer obilaze cijelo stablo i kad to nije potrebno.

Grafove možete implementirati na dva načina:

  • Pomoću nizova za popis čvorova i matrica za spremanje veza
  • Pomoću liste za popis čvorova i liste za popis susjeda

Prilikom rješavanja kolokvija, koristit ćete onaj način koji vam više odgovara.

Zadaci

Kolokvij se sastoji od 5 zadataka, okvirnog sadžaja:

  1. Stablo - unos/ispis
  2. Brisanje traženog čvora ako je list i izvršavanje neke akcije ako nije.
  3. Pretraga stabla (po ključu ili po ostalim vrijednostima) ili računanje visine.
  4. Graf - unos/ispis
  5. Pretrage (DFS/BFS).

Za prolaznu ocjenu je potrebno riješiti cijele 2 točke od 5.

Predložak izbornika

Predložak možete preuzeti sa stranice kolegija.

Predložak sadrži datoteku BTPrint.cs u kojoj se nalaze klase i metode koje se koriste za crtanje stabla. Polja koja su važna kako bi crtanje ispravno radilo:

  • lijevo, desno, BrojID

Prema tome, nemojte ta polja mijenjati. Polje BrojID uvijek označava broj na kojem se temelji uređaj (neovisno o tome radi li se konkretno o broju registracije auta ili broju osobne iskaznice i sl.). Ostale informacije koje se spremaju u čvor stabla možete slobodno dodati jer se ne crtaju.

Preporuka je da se za čvor grafa koristi posebna klasa. Također, nije loše odvojiti unos za graf od unosa za stablo jer se može dogoditi pogreška pri radu s grafovima, pa ne želite probleme s cijelim projektom.

Premda ima dosta linija u programu, također se dosta toga i ponavlja pa se lako može kopirati i prepraviti.

📌 Ovdje je svako rješenje napisano na jedan način, nije nužno najbolji niti najkraći.

Opis zadatka

a) Učitavanje podataka

Iz datoteke cvorovi.txt treba učitati podatke o vlasnicima brodova u uređeno binarno stablo. Podaci o svakom brodu sastoje se od:

  • imena
  • jedinstvenog broja registracije

Uređaj u stablu se temelji na tom broju registracije.

b) Rad s greškama (duplim podacima)

Nisu dozvoljeni duplikati pa će se spremiti zadnji koji se pojavljuje u datoteci, a ispisat će se imena duplikata koji nisu upali u stablo.

c) Ispis

Napisati metodu PostIspis() unutar klase BinarnoStablo koja ispisuje stablo (broj i naziv) po postorder poretku, neka su broj i ime odvojeni crticom: "broj-ime".

Prijedlog rješenja

Potrebno je koristiti predložak u kojem je već definirana klasa Cvor.

public string ime;

U klasi BinarnoStablo pišemo sva polja i metode koje su nam potrebne za implementaciju strukture:

  • Polje korijen tipa Cvor
  • Metode za dodavanje, brisanje, traženje, ...

Kako napisati metodu Dodaj()? To možemo na dva načina:

  • Pomoću rekurzije
  • Bez rekurzije

U ovom primjeru ćemo rješavati bez rekurzije.

Napisat ćemo metodu Dodaj() koja:

  • Prima čvor stabla
  • Vraća poruku (string) o tome što se dogodilo

Kako bi malo brže riješili, u nastavku imate primjer metode koja ne vraća ništa.

Napišite novu metodu (ili promijenite postojeću) tako da:

  • Vraća prazan string, ako je unos čvora bio uspješan
  • Vraća poruku ako je uneseni čvor duplikat (ispisati poruku u skladu sa zadatkom)

public void Dodaj(Cvor novi)
{
    if (korijen == null)
    {
        korijen = novi;
        return;
    }

    bool traziPoziciju = true;
    Cvor temp = korijen;
    while (traziPoziciju)
    {
        if (novi.BrojID < temp.BrojID)
        {
            if (temp.lijevo == null)
            {
                temp.lijevo = novi;
                traziPoziciju = false;
            }
            else
                temp = temp.lijevo; // idi dalje
        }
        else if (novi.BrojID > temp.BrojID)
        {
            if (temp.desno == null)
            {
                temp.desno = novi;
                traziPoziciju = false;
            }
            else
                temp = temp.desno; // idi dalje
        }
        else
        {
            // jednaki
            traziPoziciju = false;
        }
    }
} // Dodaj

private void TraziRek(Cvor k, int z, List<int> trazeni)
{
    if (k != null)
    {
        TraziRek(k.lijevo, z, trazeni);
        if (k.BrojID % 10 == z)
            trazeni.Add(k.BrojID);
        TraziRek(k.desno, z, trazeni);
    }
}

MojeStablo.cs

public string Brisi(int x)
{
    string poruka = "";

    // ako je prazno
    if (korijen == null)
    {
        poruka = "prazno";
        return poruka;
    }

    // postavi početne vrijednosti
    // korijen ne smijemo micati pa koristimo temp
    Cvor temp = korijen;
    Cvor roditelj = korijen;
    // ovdje umjesto bool varijable koristimo string
    // želimo znati je li korijen, lijevo (L) ili desno (D) dijete
    string strana = "korijen";

    while (temp != null)
    {
        // provjeri je li traženi nađen
        if (temp.BrojID == x)
        {
            // pobriši ako je list
            if (temp.lijevo == null && temp.desno == null)
            {
                poruka = "pobrisan: " + x;
                if (strana == "korijen")
                {
                    // pobriši
                }
                else if (strana == "L")
                {
                    // pobriši
                }
                else if (strana == "D")
                {
                    // pobriši
                }
                else
                {
                    poruka = "greška";
                }
                return poruka;
            } //list
            else
            {
                // vratiti poruku, npr.
                // Ne mogu brisati 423. Roditelj: 484
                return poruka;
            }
        }
        // ako nije nađen pomakni ga lijevo
        else if (x < temp.BrojID)
        {
            strana = "L";
            roditelj = temp;
            temp = temp.lijevo;
        }
        else
        {
            // dovršite
        }

    }

    poruka = $"Čvor {x} ne postoji";
    return poruka;
} // Brisi

Opis zadatka

Za čvorove grafa koristiti iste podatke iz datoteke cvorovi.txt, a veze su u datoteci veze.txt. Napisati klasu/strukturu Graf koja omogućuje unos čvorova i veza u neusmjeren graf.

Učitati sve podatke u strukturu grafa i ispisati na konzolu popis čvorova i veza među njima.

Ispisati graf na način da se ispiše čvor s popisom susjeda ispod njega (pazite da bude pregledno).

Prijedlog rješenja

Zadatak možemo riješiti pomoću matrice ili pomoću liste susjeda. U ovom primjeru prikazat će se rješenje (jedno od mogućih) pomoću liste susjeda.

Potrebno je definirati klasu za graf te pripradajuće pomoćne klase. Premda se graf sastoji od vrhova i bridova, možemo ih za potrebe kolokvija nazivati čvorovima i vezama kako bi imali manje pojmova kod rješavanja.

Klasa CvorGrafa:

Graf.cs

class CvorGrafa
{
    public int broj;
    public string ime;
    public bool posjecen;
    public List<CvorGrafa> susjedi;

    public CvorGrafa(int br)
    {
        broj = br;
        susjedi = new List<CvorGrafa>();

Klasa nije još dovršena jer nedostaju susjedi, no pokušajmo najprije pripremiti osnovni dio za klasu Graf. Klasa ima:

  • Polje cvorovi koje koristimo za spremanje čvorova grafa. Ovdje je u obliku liste kako bi mogli dinamički dodavati elemente. Ne zaboravite instancirati polje cvorovi!
  • Metodu DodajCvor() - koristi se za dodavanje čvorova u polje cvorovi. Pri tome moramo paziti na duple vrijednosti (moramo biti u mogućnosti dohvatiti svaki čvor na jedinstven način). Metoda može primati vrijednosti ili čvor.

Metode za dodavanje

Kod dodavanja čvorova i susjeda, potrebno je uvijek provjeriti postoji li taj čvor ili ne. Premda postoje neke ugrađene metode za provjeru (obzirom da koristimo gotovu klasu List<>), napisat ćemo vlastitu metodu jer će nam koristiti za dvije stvari:

  • Provjeru postoji li
  • Vraćanje indeksa

Neka metoda TraziIndeks() prima broj kojeg tražimo, a vraća natrag indeks na kojem se mjestu nalazu i listi ili -1 ako ga nema.

Graf.cs

// klasa Graf
public int TraziIndeks(int id)
{
    for (int i = 0; i < cvorovi.Count; i++)
    {
        if (cvorovi[i].broj == id)
            return i;
    }

    // ako ga nema
    return -1;
}

Obzirom da smo se odlučili na listu susjeda, onda je potrebno naprabiti metodu koja će se brinuti oko dodavanja susjeda (veza) unutar same klase CvorGrafa.

Opis zadatka

Ispisati koji čvor ima najviše susjeda.

Počevši od drugog upisanog čvora ispisati sve čvorove na konzolu pomoću algoritma pretrage po širini (ispisati naziv i broj).

Neposjećeni susjedi se biraju po redoslijedu unosa u graf.

Prijedlog rješenja

Za pretragu po dubini koristimo strukturu stoga. Osnovna ideja je ponavljati sljedeće korake:

  • Posjeti novi čvor
  • Stavi ga na popis (stog)
  • Obradi (ispis)

dok ima elemenata u popisu.

Algoritam započinjemo određivanjem početnog čvora, a to je prema zadatku drugi upisani čvor, odnosno čvor s indeksom 1. Kako bi naša metoda mogla funkcionirati neovisno o tome koji je početni čvor, napravit ćemo da prima indeks pa je u glavnom dijelu programa možemo pozvati kako nam bude potrebno.

Bit će nam potrebna metoda koja provjerava neposjećene susjede.

✅ Obzirom da je način implementacije grafa s listom susjeda, onda je lista u klasi CvorGrafa, onda metodu za provjeru neposjećenih susjeda možemo pisati u klasi CvorGrafa.

✅ Ako bi rješavali pomoću matrice, onda su veze spremljene u matrici koja se nalazi u klasi Graf pa u tom slučaju moramo metodu za neposjećene susjede pisati unutar klase Graf.

✅ Nije obavezno pisati metodu, možemo sve pisati i u samoj metodi za pretragu, no preglednije je ako je napišemo posebno.

Metoda prolazi kroz sve susjede i vraća prvog neposjećenog ili null ako ga nema.

Graf.cs

// klasa CvorGrafa
public CvorGrafa NepSusjed()
{
    foreach (CvorGrafa s in susjedi)
    {
        if(!s.posjecen) return s;
    }

    return null;
}
